===============================
--------- Test Cases ----------
===============================

    Test Empty call:

    >>> mylist.print_sorted()
    Traceback (most recent call last):
    NameError: name 'mylist' is not defined

    Test With Positive Integer Arguments:

    >>> my_list = MyList()
    >>> my_list.append(5)
    >>> my_list.append(4)
    >>> my_list.append(3)
    >>> my_list.append(2)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]

    Test With Negative Integer Arguments:

    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(-4)
    >>> my_list.append(-1)
    >>> my_list.append(-2)
    >>> my_list.append(-3)
    >>> my_list.print_sorted()
    [-5, -4, -3, -2, -1]

    Test With Positive And Negative Integer Arguments:

    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(8)
    >>> my_list.append(5)
    >>> my_list.append(-2)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [-5, -2, 2, 5, 8]


    Test Empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

    Test Repeated Numbers:
    >>> my_list = MyList()
    >>> my_list.append(-5)
    >>> my_list.append(8)
    >>> my_list.append(5)
    >>> my_list.append(2)
    >>> my_list.append(2)
    >>> my_list.print_sorted()
    [-5, 2, 2, 5, 8]

    Test Repeated Numbers in Strings:
    >>> my_list = MyList()
    >>> my_list.append("-5")
    >>> my_list.append("8")
    >>> my_list.append("5")
    >>> my_list.append("2")
    >>> my_list.append("3")
    >>> my_list.print_sorted()
    ['-5', '2', '3', '5', '8']
